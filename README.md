#PyTorch Intent Classification Chatbot

A simple intent-based chatbot built with PyTorch and NLTK using a Bag-of-Words approach and a feed-forward neural network (MLP) for intent classification.
This project is beginner-friendly and demonstrates how traditional NLP + neural networks work together.
