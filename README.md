# PyTorch Intent Classification Chatbot

A simple intent-based chatbot built with PyTorch and NLTK using a Bag-of-Words approach and a feed-forward neural network (MLP) for intent classification. This beginner-friendly project demonstrates how traditional NLP and neural networks work together.
